# Radio Matrix React App

## Brief

Radio matrix with rows X columns

- add/remove rows
- add/remove columns
- Upload Images
- Show stats

## Tech Stack

- React (UI Library)
- Zustand (State management)
- Typescript (Language)
- Styled Compoents (Styling)
- Jest (Test runner)
- React Testing Library (Testing)
- React Transition Group (Animation)
